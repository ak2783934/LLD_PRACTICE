# Car rental system

## Functional requirement
✅ Search cars by location, brand, type, dates
✅ Book and cancel reservations
✅ Track car availability across locations
✅ CRUD for cars, branches, users, bookings
✅ Pricing model (per day/hour)

## Entities
1. Car
2. User
3. Booking
4. Branch
5. CarRentalSystem
